# Note Taking System
This is a simple Note Taking web application made using flask that does basic CRUD operations.

## Features
- User authentication
- CRUD operations
- User friendly UI
- SQLite supported

## Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQL Alchemy
- **Tools:** Git, VS Code

## Installation
- Clone the repository
- Create and activate virtual environment
- Install dependencies from requirements.txt
- Run the server

## Usage 
- Open your browser and go to http://127.0.0.1:8000/
- Create a new account or login 
- Start exploring the features (add notes, update notes, delete notes, etc)
